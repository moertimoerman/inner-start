# Current MVP Scope

## In Scope

- Landing page and pricing flow
- Supabase auth
- Stripe checkout and subscription gating
- Protected app access
- Standard playback experience
- Two standard voice options
- Low-cost pre-generated standard audio asset flow
- Setup/preferences for voice and mix

## Prepared But Not Fully Live

- Premium child-name personalization architecture
- Additional premium themes and variants
- Experimental voice generation and debugging tools
- Conversion experimentation and analytics

## Out Of Scope For Routine MVP Changes

- Full autonomous swarm behavior
- Runtime TTS for standard playback
- Broad product rewrites
- infrastructure-heavy orchestration layers
- changing approved copy or architecture without explicit approval

## Known Ambiguities

- Which current audio build outputs are canonical for both standard voices
- Which generated assets should remain as support artifacts versus product assets
- Which current landing-page claims are approved versus temporary exploration
